# About Transpario

Choosing an internship should never feel like taking a gamble.

Transpario is a community-driven initiative that documents internship opportunities in a structured, transparent, and unbiased manner.

Instead of relying on scattered reviews or incomplete information, students can understand how an internship actually works before applying.

Our focus is transparency—not judgment.

We organize publicly available information and verified community experiences so students can make informed career decisions.

---

# What We Document

Every internship listing may include information such as:

| Category | Information |
|-----------|-------------|
| Application | Application process, eligibility, deadlines |
| Selection | Tests, interviews, hiring process |
| Cost | Registration fee, training fee, certificate fee, free |
| Benefits | Stipend, mentorship, certificate, LOR |
| Learning | Projects, technologies, guidance |
| Experience | Community experiences and transparency notes |
| Duration | Internship timeline |
| Mode | Remote, Hybrid, On-site |

# Why Transpario?

Many internship opportunities lack clear information regarding:

- Application process
- Selection criteria
- Fees
- Certificates
- Stipends
- Mentorship
- Learning quality

Our goal is to organize this information so every student has access to the same level of transparency.

# Become a Contributor

We believe meaningful communities are built by passionate contributors.

As a contributor, you may help by:

- Researching internship opportunities
- Documenting verified information
- Improving existing listings
- Maintaining data quality
- Supporting community growth

Every contributor goes through a review process before joining the community.

---

# Community Standards

Every contributor is expected to:

- Communicate professionally
- Respect others
- Share factual information
- Avoid assumptions
- Maintain confidentiality
- Be willing to learn
- Work collaboratively

Professionalism is one of our core values.
